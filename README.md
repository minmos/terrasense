### Quick Overview
* **`firmware`**: contains the esp-idf firmware running on the esp32c6
* **`cad`**: contains the CAD files for the case housing the sockets, the 12V PSU, the relais/hv pcb, and an extension for that box housing the main pcb
* **`pcb`**: contains the KiCad files for the relais/hv pcb as well as the main pcb
